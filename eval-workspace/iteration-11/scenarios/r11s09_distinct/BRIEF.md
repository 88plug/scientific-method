# Claim-evaluation brief
A vendor sells a streaming analytics box: "EXACT distinct-count (not
approximate) over any stream of up to 1 billion arbitrary 16-byte ids, single
pass, using at most 1MB of working memory, no disk, deterministic." Evaluate
with the toolkit. If impossible, prove the shape and show the arithmetic; if
possible under some reading, state exactly which relaxation makes it so.
Also: empirically demonstrate the core obstruction on a small scale if you
can (a sim that shows the information requirement).
