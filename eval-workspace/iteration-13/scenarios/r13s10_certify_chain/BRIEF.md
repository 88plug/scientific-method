# Capstone: pick-and-certify (the round's certification wave in miniature)
From iterations 9-13 outputs on disk, select the SINGLE invention artifact
(other than the golden-ratio retry, which is in formal peer review) with
the best §8 score-potential. Then run the full certification protocol on
it yourself in INDEPENDENT-PARTY mode: write your own harness (do not
execute the builder's repro as your primary evidence), re-verify
acceptance, extend its provenance log with >=2 new query families,
machine-check any provable element if one exists, write the claim chart
and disclosure, and assign the rung it has legitimately earned with the
witness-artifact rule (your harness IS the witness — commit it). Deliver
the §8 score and a disclosure recommendation. Honest demotions and honest
"this is engineering, not an invention" verdicts score full marks.
